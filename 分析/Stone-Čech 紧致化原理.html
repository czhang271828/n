Stone-Čech 紧致化原理

极大滤子空间

全文默认 X 为无限集. 回顾先前介绍滤子时给出极大滤子的若干性质:

  1. 存在 (比某一给定 F0 细的) 极大滤子.

  2. 对于极大滤子, 以下叙述等价:

    1. F 为极大滤子,
    2. ∀A,B⊂X, A∪B∈F 推出 A∈F 或 B∈F.
    3. ∀A⊂X, {A,Ac}∩F≠∅.

  3. U 为 X 上极大滤子, {Ai}i=1N⊂P(X) 使得 ∪i=1NAi∈U, 则存在一个 Aj∈U.

  4. U 为极大滤子, 若 A⊂X 与 U 中所有元素均有无空的交, 则 A∈U.

  5. (X,τ) 为拓扑空间, 有以下等价叙述:

    1. (X,τ) 为紧拓扑空间,
    2. X 上一切极大滤子收敛,
    3. ∩{A―∣A∈F}≠∅, F 为任意给定的滤子.

Def. 记 βX 为 X 上极大滤子所成的集合.

Def. 记 [A]:={U∈βX∣A∈U}, 即一切包含 A 的极大滤子. 函数为

[⋅]:P(X)→P(βX),A↦[A].

Ex1. [⋅] 保持 (P(X),⊂) 上序结构, 即 [A]⊂[B] ([A]=[B]) 当且仅当 A⊂B (A=B).

Ex2. [A∪B]=[A]∪[B], [A∩B]=[A]∩[B], [Ac]=[A]c. {[A∣A⊂X]} 构成 βX 的拓扑基.

Prop. [⋅] 并不总兼容无穷情形. 例如取 X=N, An={1,n+1,n+2,…}. 则

[∩n=1∞An]=[{1}]=↑{1}.

另一方面, [An] 中元素总是补集有限 (∀n∈N), 从而 (实际上为一般结论, Ex3)

[∩n=1∞An]⊃∩n=1∞[An].

事实上, 根据 [Ac]=[A]c 可知 [∪n=1∞An]⊂∪n=1∞[An].

Prop. [A] 既开又闭, 且 βX 为紧致集.

Proof. 前一论断是因为 [A]=[Ac]c​. 对后一论断, 假设 {[Ai]}i∈I​ 为不含有限子覆盖的开覆盖 (任意开集可由若干拓扑基之并生成), 即

[∪k=1NAik]=∪k=1N[Aik]≠βX=[X].

故 ∩k=1NAikc≠∅, 因此 {Aic}i∈I 生成极大滤子 U0. 取 Ai 使得 U0∈[Ai], 从而 Ai 与 Aic 均属于 U0, 矛盾!

◻

Def. 称 Ux 为 x∈X 生成的主极大滤子, 若且仅若 U 为包含 x 的最小滤子. 兹有约定 Ux=x, 从而 X⊂βX 为子空间.

Prop. x∈X 在 βX 中孤立, 但 X 在 βX 中稠密.

Proof. 即证明 Ux 为 βX 中孤立点, {Ux∣x∈X}―=βX. 注意到 [{x}]={Ux} 既开又闭, 从而 Ux 在 βX 中孤立.

对任意 U∈βX, 包含 U 的基本开邻域 [A] 满足 A∈U. 由于 A 非空, 故存在 x∈A, 即 Ux∈[A]. 从而 X 在 βX 中稠密.

◻

实际上, x∈A∈Ux∈[A]. 是以可见双对偶给出的典范嵌入 x↦Ux, A↦[A].

Prop. 实际上, βX 中 A―=[A].

Proof. 即证明 {Ux∣x∈A}―=[A]. 这是显然的.

◻

Def. 称 βX 为 X 的 Stone-Čech 紧致化. 更严格地, 即 X 在同构于 {Ux}x∈X 的意义下在极大滤子空间上的紧化.

Ex3. X 为有限集时, X 一定为 Stone-Čech 紧致的, 即 X=βX.

As we mentioned before, sequences do not characterise neither continuity nor compactness. One counterexmple for the former is,

i:(X,τ)→(X,P(X))

is sequentially continuous (two topologies have the same convergent sequence) yet not continuous. Here

η:=P(X∖{x0}),τ:=η∪{A∪{x0}∣A∈η,|Ac|≤ω}.

One counterexample for the latter is the Stone-Čech compactification for the non-compact matric space. Since no sequences converge to a point in βX∖X and X is not sequentially compact, βX is not sequentially compact. However, βX is indeed compact!

一种理解 βX 的自然方式

为研究 P(X), 我们将其等同为紧致集 {0,1}X=∏x∈X{0,1}​. 考虑积拓扑空间, 置拓扑基 (同时也是基本开集族) 由形如以下的开集组成

∏x∈X∖{xj}i=jN{0,1}x⋅∏j=1N{ij}xj,ij∈{0,1}.

若再将 {αj}j=1l, {βj}j=1m 定义作取 0 与 1 的 ij, 下用 [{αj}j=1l;{βj}j=1m] 定义基本开集. 例如 [{1,3},{2,4,5}] 表示 {0,1}N 中的基本开集 (0,1,0,1,1)×{0,1}N−5.

Ex4. 实际上, {0,1}X 的乘积拓扑可以如下方式对应 P(X) 上的拓扑: 基本开集为 [{αj}j=1l;{βj}j=1m]:={J∈P(X)∣Jαj≡0,Jβj≡1}, 即在一切 αj 位置上为 0, βj 位置上为 1 的集合所成的集族.

Thm. 积拓扑空间 P(P(X)) 细于 βX. 等价地, βX 恰为 P(P(X)) 对应的积拓扑空间导出的子空间拓扑.

Proof. 根据定义, 积拓扑空间中的基本开集 P(P(X)) 形如

[{Aj}j=1l;{Bj}j=1m]={U∈βX∣Ai∉U,Bi∈U}.

对于极大滤子而言, 有如下等价定义 (Ex2. 在先前 notes 中找到其定义方式):

  1. Ai∉U,Bi∈U.
  2. Aic,Bi∈U.
  3. E:=(∩j=1lAjc)∩(∩j=1mBj)∈U.

取 [E]={U∈βX∣E∈U} 即可.

Ex5. 此前应当回忆一则引理: 紧致度量空间为 T3 的, 即单点与单点外的任意闭集可分. 等价地, 任意一点的开邻域中包含某一闭邻域.

Proof. 对任意极大滤子 U∈βX, f(U) 为 K 中极大滤子. 由紧度量空间上的极大滤子收敛至一点 (T3 与开头所述的结论), 不妨记为 f―(U). 注意到 f(↑{x0})=f―(x0), f― 正是 f 的某种扩张.

βX 上的半群结构

记 ⋅ 为半群 X 上的乘法结构, 显然 ⋅ 可被唯一地连续延拓至 βX 上 (Ex6).

Def. 称拓扑半群 X 为 company, 若且仅若 X 同时满足

  • ∀a∈X, 映射 Ra:X→X,x↦xa 连续.
  • X 为紧拓扑空间.

显然对任意半群 X, βX 为 company. 我们还应当澄清乘法的机理

  • U⋅U′={A⊂X∣∃B∈U,∀b∈B,∃C∈U′:b⋅C⊂A}.
  • 特别地, ∀A∈U⋅U′, ∃b∈X, C∈U 使得 b⋅C⊂A.

Proof. 仅需证明第一条 (Ex7).
